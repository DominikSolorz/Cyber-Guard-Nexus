# Margonem-like — Strona gry (public/)

Repo zostało zastąpione prostą, statyczną wersją prototypu gry "Margonem-like" umieszczoną w katalogu `public/`.

Jak uruchomić lokalnie:
1. Otwórz plik `public/index.html` w przeglądarce (możesz użyć prostego serwera, np. `npx http-server public` lub `python -m http.server` w katalogu public).
2. Gra działa zarówno na komputerze, jak i na telefonie (dotyk): dotknij/click, aby poruszać się; przyciski ekranowe do ataku i umiejętności.

Hosting:
- Katalog `public/` jest przygotowany do hostingu statycznego (GitHub Pages, Netlify itp.).

Uwaga:
- Gra jest prototypem, używa Phaser.js z CDN, nie wymaga budowania.
- Jeśli chcesz, mogę dodać automatyczny deploy do GitHub Pages.
